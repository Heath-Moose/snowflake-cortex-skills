# Snowflake Cortex Skills

32 skills for Claude Code covering Cortex Agents, AI Functions, Cost Intelligence, Data Governance, ML, Dashboards, and more.

## Install

### Cowork (Claude Code Desktop)

1. Download `snowflake-skills.plugin` from the [latest release](https://github.com/Heath-Moose/snowflake-cortex-skills/releases/latest)
2. Open Cowork → Settings → Plugins → Import
3. Select the `.plugin` file

### CLI

```bash
claude plugin marketplace add Heath-Moose/snowflake-cortex-skills
claude plugin install snowflake-cortex-skills@snowflake-cortex-skills
```
